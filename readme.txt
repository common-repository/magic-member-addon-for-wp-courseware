This plugin adds integration between Magic Members and WP Courseware which allows you to associate course(s) to membership levels for automatic enrollment.

== Description ==
[Fly Plugins](https://flyplugins.com) presents [Magic Members](https://www.magicmembers.com/) for [WP Courseware](https://flyplugins.com/wp-courseware).

= Would you like to sell an online course with Magic Members? =
The Magic Members Addon for WP Courseware will add full integration with WP Courseware. Simply assign WP Courseware course(s) to a Magic Members membership level. When a student purchases the membership level, they will automatically be enrolled into the associated course(s).

With this addon, you will be able to create a fully automated [Learning Management System](https://flyplugins.com/wp-courseware) and sell online courses.

= Magic Members Plugin Integration with WP Courseware =
[youtube https://www.youtube.com/watch?v=UIWN8WxPfdk]

= Basic Configuration Steps =
1. Create a course with WP Courseware and add module(s), units, and quiz(zes)
2. Create a course outline page using [shortcode]
3. Create a membership level and set a price
4. Associate one or more WP Courseware courses with the membership level
5. New student pays for the membership level, and WP Courseware enrolls them to the appropriate course(s) based on the purchased membership level

== Installation ==

1. Upload the `Magic Members for WP Courseware addon` folder into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
